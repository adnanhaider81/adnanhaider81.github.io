## Update The Site

After editing files locally:

```bash
git add .
git commit -m "Update research blog"
git push
```

GitHub Pages will rebuild automatically after the push.

## Files

- `index.html` - site structure
- `styles.css` - visual design
- `script.js` - article archive and repository cards
- `assets/` - images
- `posts/` - draft space for future posts
- `.nojekyll` - direct static-file serving for GitHub Pages

## Notes

Keep public text focused on dated research notes, repository updates, methods, tutorials, and profile links.
